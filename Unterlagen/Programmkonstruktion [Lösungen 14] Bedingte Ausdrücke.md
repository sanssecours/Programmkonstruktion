# Aufgaben

## Übung

Wie lautet das Ergebnis der folgenden Ausdrücke?

```java
1 == 2 ? 'a' : 'o'                    // Wert: 'o'
(2 * 3 < 7) ? 1 : (2 + 4)             // Wert: 1
2 * 3 < 7 ? 1 : 2 + 4                 // Wert: 1
1 != 1 ? (2 != 2 ? "Leo" : "nardo") :
         (5 != 6 ? "Di" : "Caprio")   // Wert: "Di"
```
