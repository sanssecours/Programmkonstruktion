# Aufgaben

```java
static int maximum (int number1, int number2) {
    if (number1 >= number2) {
      return number1;
    }
    return number2;
}
```

- **Name der Funktion**: `maximum`
- **Parameter**:
  - Anzahl: 2
  - Name(n): `number1`, `number2`
  - Type(n): `int`, `int`
- Was macht die Funktion?

  Die Funktion retuniert das Maximum der Zahlen `number 1` und`number2`.
