# Aufgaben

- Welche der folgenden Ausdrücke sind „korrekt“ (ergeben ein Wert)? Wie lautet der Wert bzw. wo liegt das Problem?

  ```java
  true - 10
  20 / 10
  22 / 10
  "Hello" + "Bla" + 2
  true && false && (true || false)
  true + "false"
  "false" + true
  'hi' + "ho"
  2 * 3 + 4

  /* Annahme:
      - int zehn = 10;
      - double pi = 3.1415;
  */
  zehn + zehn
  "zehn" + "zehn"
  zehn / "zehn"
  zehn + " Zehen"
  "Zehen Anzahl: " + zehn
  zehn / 3
  zehn / pi
  ```
